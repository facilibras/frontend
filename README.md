# Facilibras - Frontend

Facilibras é uma aplicação web para ensino e prática de Libras (Língua Brasileira de Sinais), desenvolvida com React, TypeScript e Vite.

## Funcionalidades

- Exercícios interativos de Libras (Alfabeto, Números, Alimentos, etc)
- Reconhecimento de gestos via webcam
- Dashboard com progresso do usuário
- Ranking de usuários
- Upload e análise de vídeos para reconhecimento de sinais

## Tecnologias Utilizadas

- **React**: Interface de usuário
- **TypeScript**: Tipagem estática
- **Vite**: Build e desenvolvimento rápido
- **Zustand**: Gerenciamento de estado global
- **TanStack Router**: Navegação entre páginas
- **Lucide React**: Ícones SVG
- **Axios**: Requisições HTTP
- **TailwindCSS**: Estilização rápida e responsiva

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/facilibras/frontend.git
   cd frontend
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Como Usar

- Acesse `http://localhost:5173` no navegador.
- Escolha um exercício e siga o tutorial.
- Use a webcam para realizar e enviar gestos.
